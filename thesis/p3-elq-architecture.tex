\caption{
  The overall library architecture, which shows how the library is divided into two parts; the core and plugins. 
  The core is only accesible through the plugin \textsc{api}, and is not a part of the public \textsc{api}.
  Plugins may have interplugin \textsc{api}'s and dependencies.
  The bigger part of the public \textsc{api}'s is defined by the plugins.
}
